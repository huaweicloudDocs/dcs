# 备份指定实例<a name="CopyInstance"></a>

## 功能介绍

备份指定的缓存实例。

>![](public_sys-resources/icon-note.gif) **说明：** 
>只有主备和集群类型的缓存实例支持备份恢复操作，单机实例不支持备份恢复操作。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=CopyInstance)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS CopyInstance](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=CopyInstance)。

## URI
