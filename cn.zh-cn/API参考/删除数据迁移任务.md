# 删除数据迁移任务<a name="DeleteMigrationTask"></a>

## 功能介绍

删除数据迁移任务。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=DeleteMigrationTask)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS DeleteMigrationTask](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=DeleteMigrationTask)。

## URI
