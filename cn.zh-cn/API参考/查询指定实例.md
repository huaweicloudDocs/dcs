# 查询指定实例<a name="ShowInstance"></a>

## 功能介绍

通过实例ID查询实例的详细信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ShowInstance)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ShowInstance](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ShowInstance)。
