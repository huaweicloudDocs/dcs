# 获取日志下载链接<a name="CreateRedislogDownloadLink"></a>

## 功能介绍

获取日志下载链接。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=CreateRedislogDownloadLink)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS CreateRedislogDownloadLink](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=CreateRedislogDownloadLink)。

## URI

POST /v2/\{project\_id\}/instances/\{instance\_id\}/redislog/\{id\}/links
