# 查询实例诊断任务列表<a name="ListDiagnosisTasks"></a>

## 功能介绍

查询指定缓存实例诊断任务列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListDiagnosisTasks)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListDiagnosisTasks](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListDiagnosisTasks)。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/diagnosis
