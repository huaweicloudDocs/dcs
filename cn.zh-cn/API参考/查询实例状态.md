# 查询实例状态<a name="ListNumberOfInstancesInDifferentStatus"></a>

## 功能介绍

查询该租户在当前区域下不同状态的实例数。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListNumberOfInstancesInDifferentStatus)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListNumberOfInstancesInDifferentStatus](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListNumberOfInstancesInDifferentStatus)。
