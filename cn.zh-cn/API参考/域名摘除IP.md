# 域名摘除IP<a name="DeleteIpFromDomainName"></a>

## 功能介绍

将只读副本的IP从域名中摘除，摘除成功后，只读域名不会再解析到该副本IP。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=DeleteIpFromDomainName)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS DeleteIpFromDomainName](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=DeleteIpFromDomainName)。

## URI

DELETE /v2/\{project\_id\}/instances/\{instance\_id\}/groups/\{group\_id\}/replications/\{node\_id\}/remove-ip
