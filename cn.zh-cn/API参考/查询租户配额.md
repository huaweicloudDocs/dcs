# 查询租户配额<a name="ZH-CN_TOPIC_0166889657"></a>

## 功能介绍<a name="section164151825713"></a>

查询租户默认可以创建的实例数和总内存的配额限制，以及可以申请配额的最大值和最小值。不同的租户在不同的区域配额可能不同。
