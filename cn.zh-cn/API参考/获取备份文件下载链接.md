# 获取备份文件下载链接<a name="ListBackupFileLinks"></a>

## 功能介绍

获取指定实例的备份文件下载链接，下载备份文件。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListBackupFileLinks)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListBackupFileLinks](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListBackupFileLinks)。

## URI

POST /v2/\{project\_id\}/instances/\{instance\_id\}/backups/\{backup\_id\}/links
