# 重启实例或清空数据<a name="RestartOrFlushInstances"></a>

## 功能介绍

重启运行中的DCS缓存实例。

清空Redis4.0/Redis5.0的实例数据，数据清空后，无法撤销，且无法恢复，请谨慎操作。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=RestartOrFlushInstances)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS RestartOrFlushInstances](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=RestartOrFlushInstances)。

## URI

## 请求示例

```
PUT https://{dcs_endpoint}/v2/{project_id}/instances/status

{
  "instances" : [ "2e803f66-fbb0-47ad-b6cb-fb87f5bed4ef" ],
  "action" : "restart"
}
```

## 响应示例

**状态码： 200**

重启实例或清除实例数据成功。

```
{
  "results" : [ {
    "instance" : "e3a7019c-8824-4c1a-8289-5300f19b9f64",
    "result" : "success"
  } ]
}
```
