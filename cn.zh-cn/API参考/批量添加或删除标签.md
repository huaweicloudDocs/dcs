# 批量添加或删除标签<a name="BatchCreateOrDeleteTags"></a>

## 功能介绍

为指定实例批量添加标签，或批量删除标签。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=BatchCreateOrDeleteTags)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS BatchCreateOrDeleteTags](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=BatchCreateOrDeleteTags)。

## URI

POST /v2/\{project\_id\}/dcs/\{instance\_id\}/tags/action
