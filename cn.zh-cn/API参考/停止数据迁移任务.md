# 停止数据迁移任务<a name="StopMigrationTask"></a>

## 功能介绍

停止数据迁移任务。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=StopMigrationTask)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS StopMigrationTask](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=StopMigrationTask)。

## URI
