# 删除实例<a name="DeleteSingleInstance"></a>

## 功能介绍

删除指定的缓存实例，释放该实例的所有资源。

\[\> 如果是删除按需资源，请按照本章节执行；如果是删除包周期资源，即退订，请参考\[退订包周期资源\]\[https://support.huaweicloud.com/api-oce/zh-cn\_topic\_0082522030.html\#section2\)。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=DeleteSingleInstance)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS DeleteSingleInstance](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=DeleteSingleInstance)。
