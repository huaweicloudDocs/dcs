# 批量删除实例<a name="BatchDeleteInstances"></a>

## 功能介绍

批量删除多个缓存实例。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=BatchDeleteInstances)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS BatchDeleteInstances](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=BatchDeleteInstances)。

## URI

DELETE /v2/\{project\_id\}/instances

## 请求示例

```
DELETE https://{dcs_endpoint}/v2/{project_id}/instances?all_failure={all_failure}

{
  "instances" : [ "54602a9d-5e22-4239-9123-77e350df4a34", "7166cdea-dbad-4d79-9610-7163e6f8b640" ]
}
```

**状态码： 200**

批量删除缓存实例成功。

    "result" : "success",
