# 查询单个实例标签<a name="ShowTags"></a>

## 功能介绍

通过实例ID查询标签。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ShowTags)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ShowTags](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ShowTags)。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/tags

## 请求示例

```
POST https://{dcs_endpoint}/v2/{project_id}/instances/{instance_id}/tags
```

## 响应示例

**状态码： 200**

查询实例标签成功。

```
{
  "tags" : [ {
    "value" : "a",

    "key" : 2
  } ]
}
```
