# 查询运行中实例的统计信息<a name="ListStatisticsOfRunningInstances"></a>

## 功能介绍

查询当前租户下处于“运行中”状态的缓存实例的统计信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListStatisticsOfRunningInstances)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListStatisticsOfRunningInstances](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListStatisticsOfRunningInstances)。

## URI

GET /v2/\{project\_id\}/instances/statistic
