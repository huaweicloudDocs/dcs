# 查询产品规格<a name="ListFlavors"></a>

## 功能介绍

在创建缓存实例时，需要配置订购的产品规格编码（spec\_code），可通过该接口查询产品规格，查询条件不选时默认查询全部。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListFlavors)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListFlavors](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListFlavors)。

## URI

GET /v2/\{project\_id\}/flavors
