# 查询租户配额<a name="ShowQuotaOfTenant"></a>

## 功能介绍

查询租户默认可以创建的实例数和总内存的配额限制，以及可以申请配额的最大值和最小值。不同的租户在不同的区域配额可能不同。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ShowQuotaOfTenant)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ShowQuotaOfTenant](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ShowQuotaOfTenant)。

## URI

GET /v2/\{project\_id\}/quota

