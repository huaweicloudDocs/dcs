# 创建实例诊断任务<a name="CreateDiagnosisTask"></a>

## 功能介绍

诊断指定的缓存实例。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=CreateDiagnosisTask)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS CreateDiagnosisTask](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=CreateDiagnosisTask)。

## URI

POST /v2/\{project\_id\}/instances/\{instance\_id\}/diagnosis

```
}
```
