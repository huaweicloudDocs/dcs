# 查询慢日志<a name="ListSlowlog"></a>

## 功能介绍

查询慢日志。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListSlowlog)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListSlowlog](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListSlowlog)。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/slowlog

## 请求示例

```
GET https://dcs.cn-south-1.myhuaweicloud.com/v2/a4d31cb6-3d72-4fdc-8ec9-6e3a41e47f71/instances/5560df16-cebf-4473-95c4-d1b573c16e79/slowlog?start_time=1598803200000&end_time=1599494399000
```

## 响应示例

**状态码： 200**

查询慢日志成功。

```
{
  "count" : 1,
  "slowlogs" : [ {
    "id" : 1,
    "start_time" : "2020-06-19T07:06:07Z",
    "command" : "SLOWLOG GET 128",
    "duration" : "17",
    "shard_name" : "group-0"
  } ]
}
```

**状态码： 400**

非法请求。

```
{
  "error_code" : "DCS.4800",
  "error_msg" : "Invalid start_time in the request."
}
```
