# 查询大key分析任务列表<a name="ListBigkeyScanTasks"></a>

## 功能介绍

查询大key分析任务列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListBigkeyScanTasks)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListBigkeyScanTasks](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListBigkeyScanTasks)。

## URI
