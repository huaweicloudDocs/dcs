# 查询在线迁移进度明细<a name="ShowMigrationTaskStats"></a>

## 功能介绍

查询在线迁移进度明细。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ShowMigrationTaskStats)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ShowMigrationTaskStats](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ShowMigrationTaskStats)。

## URI

GET /v2/\{project\_id\}/migration-task/\{task\_id\}/stats

## 响应示例

**状态码： 200**

查询在线迁移进度明细成功。

```
{
  "full_migration_progress" : 100,
  "offset" : 100,
  "source_dbsize" : 531221,
  "target_dbsize" : 531211,
  "target_input_kbps" : 50,
  "target_ops" : 5300,
  "is_migrating" : true
}
```

**状态码： 400**

非法请求。

```
{
  "error_code" : "DCS.4133",
  "error_msg" : "Instance job is not found."
}
```
