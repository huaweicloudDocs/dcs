# 设置备节点优先级<a name="UpdateSlavePriority"></a>

## 功能介绍

设置副本优先级，主节点故障时，权重越小的备节点切换为主节点的优先级越高。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=UpdateSlavePriority)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS UpdateSlavePriority](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=UpdateSlavePriority)。

## URI

POST /v2/\{project\_id\}/instances/\{instance\_id\}/groups/\{group\_id\}/replications/\{node\_id\}/slave-priority
