# 创建热key分析任务<a name="CreateHotkeyScanTask"></a>

## 功能介绍

创建热key分析任务，Redis 3.0 不支持热key分析。

热key分析需要将缓存实例配置参数maxmemory-policy设置为allkeys-lfu或volatile-lfu。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=CreateHotkeyScanTask)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS CreateHotkeyScanTask](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=CreateHotkeyScanTask)。

## URI

POST /v2/\{project\_id\}/instances/\{instance\_id\}/hotkey-task
