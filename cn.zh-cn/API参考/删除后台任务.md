# 删除后台任务<a name="DeleteBackgroundTask"></a>

## 功能介绍

删除后台任务

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=DeleteBackgroundTask)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS DeleteBackgroundTask](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=DeleteBackgroundTask)。

## URI

DELETE /v2/\{project\_id\}/instances/\{instance\_id\}/tasks/\{task\_id\}
