# 查询实例价格<a name="ZH-CN_TOPIC_0141931512"></a>

查询实例当前价格。

## **URI**<a name="section678380145219"></a>

GET /v1.0/products/prices

## **请求消息**<a name="section525620116529"></a>

**请求参数**

无。

**请求示例**

```
GET https://{dcs_endpoint}/v1.0/products/prices
```
