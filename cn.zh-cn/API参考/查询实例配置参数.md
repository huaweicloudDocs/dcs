# 查询实例配置参数<a name="ListConfigurations"></a>

## 功能介绍

查询指定实例的配置参数信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListConfigurations)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListConfigurations](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListConfigurations)。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/configs
