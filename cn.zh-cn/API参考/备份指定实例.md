# 备份指定实例<a name="ZH-CN_TOPIC_0066959097"></a>

>![](public_sys-resources/icon-note.gif) **说明：**   
>只有主备类型的缓存实例支持备份恢复操作，单机和集群实例不支持备份恢复操作。
