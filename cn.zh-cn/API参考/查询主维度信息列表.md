# 查询主维度信息列表<a name="ListMonitoredObjects"></a>

## 功能介绍

查询主维度对象列表，主维度ID当前支持dcs\_instance\_id，dcs\_memcached\_instance\_id。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListMonitoredObjects)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListMonitoredObjects](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListMonitoredObjects)。

## URI

GET /v2/\{project\_id\}/dims/monitored-objects

```
GET https://{dcs_endpoint}}/v2/{project_id}}/dims/monitored-objects?dim_name={dim_name}
```
