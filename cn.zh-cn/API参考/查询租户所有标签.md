# 查询租户所有标签<a name="ListTagsOfTenant"></a>

## 功能介绍

查询租户在指定Project中实例类型的所有资源标签集合。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListTagsOfTenant)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListTagsOfTenant](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListTagsOfTenant)。

## URI

GET /v2/\{project\_id\}/dcs/tags

## 请求示例

```
POST https://{dcs_endpoint}/v2/{project_id}/dcs/tags
```

## 响应示例

**状态码： 200**

查询租户所有标签成功。

```
{
  "tags" : [ {
    "value" : [ "value1", "value2" ],
    "key" : 1
  }, {
    "value" : [ "value1", "value2" ],
    "key" : 2
  } ]
}
```
