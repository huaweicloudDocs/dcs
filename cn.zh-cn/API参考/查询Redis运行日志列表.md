# 查询Redis运行日志列表<a name="ListRedislog"></a>

## 功能介绍

查询Redis运行日志列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListRedislog)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListRedislog](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListRedislog)。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/redislog

## 请求示例

```
GET https://dcs.cn-south-1.myhuaweicloud.com/v2/a4d31cb6-3d72-4fdc-8ec9-6e3a41e47f71/instances/5560df16-cebf-4473-95c4-d1b573c16e79/redislog?log_type=run&offset=1&limit=10
```

**状态码： 200**

查询运行日志成功。

```
{
  "total_num" : 1,
  "file_list" : [ {
```
