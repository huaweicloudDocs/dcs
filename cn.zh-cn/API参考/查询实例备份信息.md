# 查询实例备份信息<a name="ListBackupRecords"></a>

## 功能介绍

查询指定缓存实例的备份信息列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListBackupRecords)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListBackupRecords](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListBackupRecords)。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/backups
