# 查询实例恢复记录<a name="ListRestoreRecords"></a>

## 功能介绍

查询指定缓存实例的恢复记录列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=DCS&api=ListRestoreRecords)中调试该接口，并获取该接口关联的CLI示例：  [hcloud DCS ListRestoreRecords](https://apiexplorer.developer.huaweicloud.com/apiexplorer/cli?product=DCS&api=ListRestoreRecords)。

## URI

GET /v2/\{project\_id\}/instances/\{instance\_id\}/restores
